onTabTraversal

open fun onTabTraversal(): Boolean

Overrides View.onTabTraversal

Called when tab was pressed and it was determined that this is the resultant View of the traversal. It should set the current state to HOVER. Returns whether or not this is the last View in a potential chain of Views.
