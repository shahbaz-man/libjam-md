NinePatchDrawable(ninepatch: BufferedImage, doCache: Boolean = false, drawMiddlePatch: Boolean = true)

Accepts a standard Android nine-patch foreground and draws it as a nine-patch drawable. NOTE: this is slow, so using caching (by setting cache = true) if you are going to be constantly be repainting the foreground. This may improve performance just a bit.
