AccelerateDecelerateInterpolator

object AccelerateDecelerateInterpolator : Interpolator

An interpolator where the rate of change starts and ends slowly but accelerates through the middle.

Functions

getValue fun getValue(input: Float): Float

Interpolates the given value (between 0.0 - 1.0) to the interpolated value between (0.0 - 1.0). The interpolated value should always give an output of 1.0 for the input of 1.0.
