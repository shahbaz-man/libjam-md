View

open class View

An open class that defines a View. A View is simply a lightweight widget which all other widgets originate from. It implements the bare minimum functionality, allowing for easy extensibility.

Properties

background var background: Drawable?

The background drawable of this view. Defaults to the THEME background color.
layoutSize var layoutSize: FloatDim

The size of this view set by the layout. Will be between min and max size once a layout pass has been conducted.
maxSize var maxSize: FloatDim

The max and minimum sizes of this view. Consider the maximum size as the preferred size as the layout will always try to grow the view to that size.
minSize var minSize: FloatDim
visibility var visibility: Int

Defines the visibility of this view. May either be VISIBLE or INVISIBLE
